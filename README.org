* 简介

这是一个用 [[https://www.haskell.org][Haskell]] 编写的JSON解析系统，包括JSON源文件的解析，格式化和外部语言转化。解析器依照规范 [[https://www.json.org/][ECMA-404]] 编写，部分功能正在完善。这个系统的目标是实现一个可扩展的，包括解析、格式化和外部语言接口的JSON分析库。
